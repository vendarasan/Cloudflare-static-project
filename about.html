About this template

Nova Static is a lightweight, accessible starter with zero dependencies. Use it for landing pages, portfolios, or documentation.

Semantic

Headings, landmarks, and contrast-friendly colors support accessibility.

Customizable

Change the palette via :root CSS variables. Add sections with ready-made utility classes.

Portable

Upload to any static host: GitHub Pages, Netlify, Vercel, or your own server.

File structure

Organized and tidy so you can find things quickly.
